# Contributing

## Unconventional Conventional Commits

### Adding

```txt
add(mod/resourcepack/shaderpack): MOD_NAME/RESOURCE_PACK_NAME/SHADER_PACK_NAME

VERSION: MOD_VERSION/RESOURCE_PACK_VERSION/SHADER_PACK_VERSION
```

### Configuration

```txt
config(mod/resourcepack/shaderpack): MOD_NAME/RESOURCE_PACK_NAME/SHADER_PACK_NAME

CHANGES
```

### Updating (Non-version Update)

```txt
change(mod/resourcepack/shaderpack): MOD_NAME/RESOURCE_PACK_NAME/SHADER_PACK_NAME

CHANGES
```
